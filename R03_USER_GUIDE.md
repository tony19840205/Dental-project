# 📘 R03 牙位不一致檢測 - 使用指南

## 🎯 功能說明
當診斷說明和處置說明中的牙位不一致時，系統會自動發出警告，避免資料錯誤。

## 📝 使用步驟

### 步驟1: 填寫診斷
1. 選擇患者（例如：P002 李美玲）
2. 選擇診斷項目（例如：輕度齲齒）
3. 點擊「🤖 診斷生成」
4. 系統生成診斷說明，包含牙位號碼
   ```
   患者主訴#34對甜食敏感，理學檢查發現齲齒...
   ```

### 步驟2: 填寫處置
1. 選擇處置項目（例如：補牙）
2. 點擊「🤖 處置生成」
3. 系統會從診斷說明中提取牙位並生成處置說明
   ```
   針對#34進行補牙，移除舊填補物...
   ```

### 步驟3: 手動修改牙位（觸發不一致）
如果您**不小心**或**故意**修改了牙位：
```
針對#33進行補牙，移除舊填補物...  ← 改成#33
```

### 步驟4: 執行檢測
點擊右上角「🔍 FHIR檢測」按鈕

### 步驟5: 查看警告
系統會顯示 🔴 ALERT 警告：

```
🔴 ALERT 
   處置和診斷的牙位不匹配
   
   處置牙位 #33 沒有對應的診斷記錄
   診斷牙位為 #34
   
   ⚠️ 注意：請檢查牙位標記或補充診斷
```

同時，相關欄位會以**亮黃色背景 + 紅色邊框**高亮顯示。

## ✅ 正確的做法

### 方法1: 修正處置牙位
將處置說明中的牙位改回與診斷一致：
```
針對#34進行補牙，移除舊填補物...  ✅
```

### 方法2: 補充診斷
如果確實要處置#33，則需要：
1. 在診斷說明中加入#33
   ```
   患者主訴#34、#33對甜食敏感...
   ```
2. 或新增第二個診斷項目

## 🔍 常見情況

### 情況1: 多顆牙齒同時處置
**診斷說明**:
```
患者主訴#34、#35對甜食敏感
```

**處置說明**:
```
分別針對#34、#35進行補牙
```
✅ 牙位一致，無警告

### 情況2: 部分牙位不匹配
**診斷說明**:
```
患者主訴#34、#35對甜食敏感
```

**處置說明**:
```
針對#33、#34進行補牙
```
⚠️ 警告：#33 無對應診斷

### 情況3: 處置記錄二
如果使用「處置記錄二」，系統也會檢查：

**診斷說明**: #34  
**處置記錄一**: #34 ✅  
**處置記錄二**: #33 ⚠️ 警告

## 🎨 視覺提示

當檢測到牙位不一致時：

1. **警告區域**: 顯示 🔴 紅色標籤
2. **診斷欄位**: 亮黃色背景 + 紅色邊框
3. **處置欄位**: 亮黃色背景 + 紅色邊框
4. **說明文字**: 詳細指出哪些牙位不匹配

## 🚨 重要提醒

1. **先診斷後處置**: 建議先生成診斷說明，再生成處置說明
2. **避免手動修改牙位**: 如需修改，請同步更新診斷和處置
3. **檢查後再存檔**: 在存檔前先執行FHIR檢測，確保無警告
4. **警告不會阻止存檔**: 系統會警告但不強制修正（未來可能改為強制）

## 📊 檢測範圍

R03 會檢測：
- ✅ 診斷說明 vs 處置說明
- ✅ 診斷說明 vs 處置說明2
- ✅ 多顆牙齒的部分不匹配
- ✅ 自動排除無牙位的診斷/處置

R03 不檢測：
- ❌ 跨就診的牙位一致性（由其他規則處理）
- ❌ 歷史記錄的牙位（由R04等規則處理）

## 💡 最佳實踐

1. **使用自動生成功能**
   - 點擊「診斷生成」和「處置生成」按鈕
   - 系統會自動確保牙位一致

2. **定期檢測**
   - 填寫完表單後立即檢測
   - 修改內容後重新檢測

3. **理解警告含義**
   - 不要忽略警告
   - 根據實際情況修正或補充
